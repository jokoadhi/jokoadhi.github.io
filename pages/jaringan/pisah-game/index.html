KONFIGURASI PISAH TRAFFIC GAME ONLINE

Diperbarui: 25 Oktober 2025

1. Apa Itu Konfigurasi Pisah Traffic Game Online dan Fungsinya Apa?

Kenapa konfigurasi pisah traffic game online perlu diterapkan di lingkungan jaringan yang kecil maupun besar?, ketika kita bermain game online disaat traffic router sedang full, biasanya akan mendapatkan latency yang cukup tinggi karena traffic game dan aplikasi lain harus berbagi sama rata, untuk itu konfigurasi pisah traffic game ini diperlukan, agar traffic game dan aplikasi lain tidak tercampur, kelebihannya latency game akan jauh lebih rendah ketimbang sebelum diterapkan konfigurasi pisah traffic game.

3. Langkah-langkah Penerapan

Masuk ke aplikasi winbox untuk me-remote mikrotiknya dan pastikan mikrotik sudah dikonfigurasi dasar dahulu, jika sudah selanjutnya masuk ke menu IP -> Firewall -> Mangle.

Tambahkan rule baru untuk menandai koneksi game onlinenya, dalam dokumentasi ini kita akan pisah traffic game Mobile Legends. Chain pilih prerouting, untuk protocol dan Dst-port menyesuaikan dengan list port game diatas.

Lalu pilih tab Action pilih mark connection, kolom New Connection Name diisi nama sesuai keinginan, klik Apply OK.

Lanjutkan ke semua port Mobile legends, yang berubah hanya port dan protocol nya menyesuaikan dengan list port game diatas. jika sudah tampilannya akan seperti gambar dibawah.

Tambahkan 1 rule lagi untuk menandai paket dari traffic Mobile Legends, Chain pilih forward, Connection mark diisi dengan rule koneksi sebelumnya yang sudah kita buat.

Selanjutnya ke tab Action pilih Mark Packet, kolom New Packet Mark diisi dengan nama sesuai keinginan, hilangkan centang Passthrough, Klik Apply OK.

Masuk ke menu Queues, untuk melakukan konfigurasi pemisahan traffic nya.

Tambahkan rule untuk traffic dari game, name diisi sesuai keinginan, Target diisi dengan IP Client yang akan dipisahkan traffic game nya, max-limit diisi sesuai kebutuhan.

pindah ke tab Advanced, Packet Mark diisi dengan rule paket mobile legends yang sebelumnya sudah kita buat, Priority diisi 1, Queue Type diisi dengan pcq-upload dan pcq-download, Klik Apply OK.

note : pastikan rule traffic game urutannya harus diatas dari semua rule client.

tambahkan lagi rule limitasi untuk client, Name diisi sesuai keinginan, Target diisi dengan IP Client, max-limit diisi sesuai kebutuhan, Klik Appy OK.

Konfigurasi sudah selesai, harusnya sampai langkah ini traffic game mobile legends sudah terpisah tidak tercampur dengan traffic client biasa seperti streaming video, download file, dll.
